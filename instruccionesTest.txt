al descargar el codigo

1. instalar dependencias

    npm install

2. En la terminal correr el test

    npm test a

3. Para ver la cobertura del test

    npm test -- --coverage a 

IMPORTANTE: si despues de ejecutar todo esto les sale el siguiente error

● Test suite failed to run

    fatal: cannot use .git/info/exclude as an exclude file

En ese caso de que les salga ese error, eliminen la carpeta ".git" y vuelvan a intentar
